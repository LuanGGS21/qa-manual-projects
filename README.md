QA Manual – Projeto de Testes de Login

Projeto prático de QA Manual com foco em testes funcionais, smoke tests e documentação de bugs.

Funcionalidades testadas
- Login de usuário
- Validação de campos
- Mensagens de erro
- Logout

Tipos de teste
- Testes funcionais
- Testes exploratórios
- Smoke tests

Ferramentas utilizadas
- Jira (simulado)
- DevTools (Chrome)
- Google Sheets / Markdown

Objetivo
Demonstrar prática em QA Manual, organização de testes e qualidade de software.
